## Setup

To setup the package.

```
npm install --global gulp
npm install
gulp build
```

To watch file changes.

```
gulp watch
```

Browser live reload.

```
gulp browser
```

## Usage

Edit ```src/App.jsx``` file and run ```gulp build```, the compiled file is at ```dist/App.js```. Please run ```gulp browser``` to load ```dist/App.html``` for resutls.
